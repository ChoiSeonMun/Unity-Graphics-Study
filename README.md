# 개요
본 스터디는 유니티 셰이더를 공부하기 위한 모임이다.

# 운영 방법
- 각 스터디원은 각 주차별로 보고서를 작성한다.
- 모임 시간은 미정이다.
- 모임에서는 해당 주차의 내용을 서로 리뷰하고, 스터디 건의 사항을 의논한다.
- 궁금한 것, 잘 이해가 가지 않는 것, 좋은 자료 등등을 적극적으로 공유한다.
- 사용되는 교재는 [유니티 쉐이더 스타트업](http://www.yes24.com/Product/Goods/46193692)이다.
